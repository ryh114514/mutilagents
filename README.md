# 结构说明

## 资源目录
- 用户上传资源：`./cache_dir/uploads`
- 网络下载资源：`./cache_dir/net_store`
- 预置资源：`./cache_dir/pre_stored`
- 生成文件：`./cache_dir/generate`

## 智能体功能
- 网络智能体：搜寻URL资源下载到指定目录并返回路径，或直接总结网页内容
- 编程智能体：合并了原代码智能体与命令执行智能体功能，便于直接编写并执行代码

## 优化要求
- 清晰描述每个智能体和MCP工具的使用方法
- 每个智能体的提示词最好先从prompts.py CV复制，并根据职责补充内容
- 每个智能体的提示词中，应包含对每个工具的详细描述，包括工具的功能、使用方法、返回值、错误处理等
