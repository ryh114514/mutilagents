##  MCP工具输入输出详表



###  推理智能体

| 工具名称         | 输入 (INPUT)                 | 输出 (OUTPUT)                |
| ---------------- | ---------------------------- | ---------------------------- |
| **逻辑推理工具** | 前提条件、推理规则、问题描述 | 推理结论、推理过程链、置信度 |
| **问题分解工具** | 复杂问题描述、分解策略       | 子问题列表、依赖关系图       |
| **决策支持工具** | 决策场景、可选方案、评估标准 | 推荐方案、风险评估、建议     |

###  网络智能体

| 工具名称            | 输入 (INPUT)                         | 输出 (OUTPUT)             |
| ------------------- | ------------------------------------ | ------------------------- |
| **通用搜索工具**    | 搜索关键词、引擎类型、结果数量、语言 | 搜索结果列表、相关性分数  |
| **API调用工具**     | API端点URL、请求方法、请求头/体      | API响应数据、状态码、时间 |
| **url资源获取工具** | URL                                  | URL对应资源               |
| **实时信息工具**    | 信息类型、位置/符号、时间范围        | 实时信息、更新时间、来源  |

###  文件智能体

| 工具名称             | 输入 (INPUT)                 | 输出 (OUTPUT)              |
| -------------------- | ---------------------------- | -------------------------- |
| **文件读取工具**     | 文件路径、编码格式、读取模式 | 文件内容、元信息、大小时间 |
| **目录列表工具**     | 目录路径、过滤模式、递归深度 | 文件列表、文件夹结构、统计 |
| **文件内容分析工具** | 文件路径、分析类型、参数设置 | 分析结果、统计信息、摘要   |

###  命令智能体

| 工具名称         | 输入 (INPUT)                         | 输出 (OUTPUT)                |
| ---------------- | ------------------------------------ | ---------------------------- |
| **命令执行工具** | 命令字符串、工作目录、环境变量、超时 | 标准输出、错误、退出码、时间 |
| **批处理工具**   | 命令列表、执行顺序、依赖关系、策略   | 每个命令结果、整体状态、错误 |

###  代码智能体

| 工具名称         | 输入 (INPUT)                     | 输出 (OUTPUT)      |
| ---------------- | -------------------------------- | ------------------ |
| **代码写入工具** | 代码字符串、编程语言、路径、名称 | 成功信息，完整路径 |

###  数据智能体

| 工具名称         | 输入 (INPUT)                     | 输出 (OUTPUT)                |
| ---------------- | -------------------------------- | ---------------------------- |
| **数据分析工具** | 数据集、分析任务、统计方法、参数 | 分析结果、统计指标、洞察发现 |



##  工作流程示例

```
用户请求 → 对话协调智能体 → 意图识别 → 智能体路由
   ↓
专业智能体（如文件智能体）→ MCP工具调用 → 执行任务
   ↓
结果返回 → 对话协调智能体 → 结果整合 → 用户响应
```
